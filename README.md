
# Low Code Chat UI
Compare multiple AI models responses between tabs. No chat history.

&nbsp;

<div align="center"><img src="preview.png" width="800"></div>

&nbsp;

## Repository Structure
```

chat-comparison/
├── aitools.py
├── chat.py
├── preview.gif
├── requirements.txt
├── setup.sh
├── start.sh
```

&nbsp;

## Run this demo locally
```
git clone https://github.com/BrianLesko/chat-comparison.git
cd chat-comparison
sudo chmod +x setup.sh
./setup.sh
./start.sh
```
